# Kawasaki Showroom Management System

## Overview

A full-stack web application for managing a Kawasaki motorcycle showroom. The system helps manage bike inventory, customer bookings, test rides, and showroom operations.

## Features

* Bike inventory management
* Test ride booking
* Admin dashboard
* Customer booking management
* Bike price prediction using Machine Learning
* Product and accessories catalog

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript
* Machine Learning

## Project Structure

* app.py
* templates/
* static/
* instance/
* requirements.txt
